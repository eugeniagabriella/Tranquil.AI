# 🌿 Tranquil.AI

A CBT-based journaling platform built during a 48-hour hackathon to help users manage social anxiety through structured reflection and AI-assisted emotional support.

---

## 💡 Inspiration

Social anxiety affects millions of people and is often left unspoken. While professional therapy exists, many individuals still hesitate to seek help due to stigma, accessibility, or fear.

Tranquil.AI was created to bridge this gap by combining **Cognitive Behavioral Therapy (CBT)** techniques with an AI-powered journaling experience that is private, structured, and supportive.

---

## ✨ What It Does

Tranquil.AI helps users reflect on their thoughts using guided CBT-based journaling:

- 🧠 **CBT Journaling Techniques**
  - Reframing negative thoughts  
  - Savoring positivity  
  - Exposure hierarchy  

- 💬 **AI Emotional Support**
  - Empathetic conversational AI  
  - Tone-aware responses  
  - Encourages self-reflection and confidence  

- 📊 **Personal Insights**
  - Mood tracking over time  
  - Key takeaways from journaling  
  - Actionable tasks and affirmations  

- 🎮 **Gamification**
  - Encourages consistency through engagement mechanics  

- 👩‍⚕️ **Therapist-Oriented Summary**
  - Structured summaries designed to help mental health professionals better understand user progress

## 🛠️ Tech Stack

---

## ⚙️ How It Was Built

- Designed in Figma before development  
- Built as a full-stack web application using Next.js  
- AI layer integrated using GPT-4-based models  
- Supabase used for database and backend services  
- Clerk used for authentication  

---

## 🚧 Challenges

With only 48 hours, the main challenge was prioritizing features and defining a clear MVP scope. The team focused on delivering a functional core experience rather than implementing all planned features.

---

## 🎯 Key Takeaways

- Shipping a working product is more important than feature completeness  
- Clear communication and prioritization are critical in fast-paced development  
- Hackathons are more about execution and learning than perfection
